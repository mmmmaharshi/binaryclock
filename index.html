How to Read This Clock

Each column (H, M, S) represents a number. To find the number, you simply add the values of the lit-up LEDs in that column. The values are shown on the far left.

For Example, to get the number 25:

32

16

8

4

2

1

You would add: 16 + 8 + 1 = 25
